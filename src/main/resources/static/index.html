How To Explain This In Interview

This is an enterprise incident management backend inspired by tools like BMC Helix. It is not just CRUD. It has lifecycle, assignment, SLA tracking, comments, timeline, audit logs, analytics, duplicate detection, and explainable root cause suggestions.

Read-only demo APIs are public for easy review. Write/admin APIs still need JWT security.

Render free tier may sleep after inactivity, so first load can be slow.
